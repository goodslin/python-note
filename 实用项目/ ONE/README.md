# python-note

ONE主要内容是每日一句话,一幅图片,一篇文章,一个问题                                                                     

目标:爬取ONE往期所有的每日一句,并保存下来                                                                            
思路:                                                                                                           
  1.由于是所有的往期,所以要先确定每期页面的url的规律                                                                    
  2.查看页面源代码,确定要爬取内容的位置                                                                                     
  3.写爬虫程序,爬取内容并保存
